Infrastructure

This page is still under construction.

I have recently been experimenting with Amazon Web Services which lead to conception of this website. It runs on the free tier setup which gives limited free access to most of the AWS services. Since I have yet to decide on a permanent set-up, the code base can be deployed to most servers with minor changes to the deploy script and is minimally tied into the AWS ecosystem. This site is built in Django 1.7 and is currently set up to run with WSGI. The server is provisioned using a fabric deploy script which, in addition to updating the code base, installs Nginx and virtualenv as well as updates the database structure. This page will eventually contain a much more detailed description of the project. In the mean time, more information can be found from the source code on GitHub.
