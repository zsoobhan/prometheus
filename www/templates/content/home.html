Ziyad Soobhan

Developer, Climber, Astrophysicist

I am a web-developer with an MSci in Astrophysics as well as an avid rock-climber. My work involves the implementation and maintenance of solutions primarily engineered in Python. which has lead to me being exposed to various mechanisms of content delivery. I work with various web technologies such as WSGI, Nginx, Apache, VirtualEnv and am particularly interested in using frameworks such Django as well as the rest of the stack. When not at work, I most probably can be found at my local climbing wall where I train three to four times a week.

Infrastructure

nginx, supervisor, uWSGI, virtualenv, django

The stack is currently set up on a DigitalOcean droplet namespaced with a hierarchical folder structure, symlinks and a virtualenv. The codebase is deployed via a fabric script which provisions the server, migrates the database, updates the configuration files and restarts the necessary services. The web application itself is managed with supervisor and uWSGI which creates a websocket accessed and served using Nginx while all of the staticfiles are pushed and served via AWS S3. For more information with regards to the status of the site, check my twitter feed or github repository, both whose links may be found at the bottom of the page.
