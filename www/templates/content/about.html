Infrastructure

nginx, supervisor, uWSGI, virtualenv, django

The current stack for this website is currently set up on a DigitalOcean droplet namespaced with a hierarchical folder structure, symlinks and a virtualenv. The codebase is deployed via a fabric script which provisions the server, migrates the database, updates the configuration files and restarts the necessary services. The web application itself is managed with supervisor and uWSGI which creates a websocket accessed and served using Nginx while all of the static files are pushed and served via AWS S3. For more information with regard to the status of the site, check my twitter feed or GitHub repository, both whose links may be found at the bottom of the page.
